  1. hexo generate时出现

Commands:

help Get help on a command.

init Create a new Hexo folder.

version Display version information.

………………………………

原因：没有在Hexo的目录下打开Git Bash，没有用npx hexo （我直接用了win的终端）


  1. 本地的localhost:4000端口被占用
    之前的一个Gridea博客把本地的4000占了，hexo要换一个，网上搜了一下没法在命令行里改（好像新版本的Hexo取消了这个功能）

解决：node_modules —> 往下狂拉找到hexo-server —> package.json —> 在里面改port端口 （我改成4004）
