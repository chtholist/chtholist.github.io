Windows快捷键

  1. 2020-03-22
  2. 92字
  3. 1 min read

Win键 +E 打开资源管理器
Win键 + Shift + S 截屏
Win键 + L 锁屏
windows键+R，输入psr.exe回车 屏幕录制
Win键 + D 返回桌面
Win键 +Ctrl+D 添加虚拟桌面
Win键 +Ctrl+F4 删除虚拟桌面
Win键 + ←/→ 左右分屏
Windows+R，输入osk 虚拟键盘

C:\Users\你的名字 windows的管理员账号文件夹
